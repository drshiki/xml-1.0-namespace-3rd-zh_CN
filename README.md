此项目的起因是因为作者在课程中使用的XML教材错漏百出以及在内容方面的质量相当不理想，所以作者在学习XML的过程中大多时候都通过XML规范等学习，刚好这份文档蝙蝠相对短小，刚好用来锻炼一下作者的翻译能力。不过最重要的还是，一时兴起。
